## ❓ Frequently asked questions

Does the software collect data?
No. The application processes all files locally on your computer.

Can I move the installed folder?
We recommend keeping the application in its default program files directory to ensure stable performance.

What happens if the power goes out?
The software pauses tasks and resumes them automatically when you restart your computer. No data loss occurs during these interruptions.

How do I uninstall naemtnu?
Go to your Windows Settings, select Apps, find naemtnu in the list, and select Uninstall. This removes all program files and the desktop shortcut.
